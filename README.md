# 🤟 Sign Language Hub: Unified ASL & Emotion Recognition

A deep learning-based real-time system for **American Sign Language (ASL) alphabet recognition** and **facial emotion detection**. The project combines computer vision and convolutional neural networks to recognize ASL hand gestures and human emotions simultaneously.

---

## 📌 Overview

This project aims to build an intelligent recognition system capable of:

- Detecting American Sign Language (ASL) hand gestures.
- Recognizing facial emotions in real time.
- Processing live webcam/video input with high accuracy.
- Providing fast inference suitable for real-world applications.

---

## ✨ Features

- 🤟 Real-time American Sign Language (ASL) alphabet recognition
- 😀 Real-time facial emotion detection
- 🎥 Live webcam support
- ⚡ Optimized for low-latency inference
- 🧠 CNN-based classification models
- 📊 Robust performance on unseen test data

---

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Haar Cascade Classifier
- Convolutional Neural Networks (CNN)

---

## 📂 Dataset

### American Sign Language (ASL)

- 34,000+ training images
- 40 gesture classes
- Image preprocessing using OpenCV

### Emotion Recognition

- Facial images processed using Haar Cascade face detection
- CNN trained for emotion classification

---

## 🧠 Model Architecture

### ASL Recognition

- Convolutional Neural Network (CNN)
- Image normalization and preprocessing
- Data augmentation
- Softmax classifier for gesture classification

### Emotion Recognition

- Haar Cascade for face detection
- CNN-based emotion classifier
- Real-time prediction pipeline

---

## ⚙️ Preprocessing Pipeline

- Image resizing
- Grayscale conversion
- Pixel normalization
- Face detection using Haar Cascade
- Data augmentation
- Batch generation for training
